## Initiate

#### npm

```
npm init @a7mooz/djs-handler@latest
```

#### yarn

```
yarn create @a7mooz/djs-handler
```

#### pnpm

```
pnpm create @a7mooz/djs-handler
```

## Get it to work

-   Put your environment variables `.env` file from the example in `.env.example`

-   `cd your-project-name`

-   Start Your Project:

##### npm

```
npm run dev
```

##### yarn

```
yarn dev
```

##### pnpm

```
pnpm dev
```
